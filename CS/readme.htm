ASPxGridView - Using GridViewDataComboBoxColumn as a lookup control

This demo implements lookup functionality using the following two techniques: - "Category LookUp" column editor populates its drop-down list from the bound CategoriesDS data source. - "RunTime LookUp" column editor is populated via code in the PageLoad event.
